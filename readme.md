Hello

this is a fun mini web app made with only HTML, CSS and javaScript. in this project, a small funny cleaner character tries to clean the screen. 
this screen has fog, dirt, drops and fingerprints. when the user moves the dirty screen gets wiped.

what is inside
foggy screen effect
dust and dirt spots
water drops
fingerprints
wipe effect with mouse
funny cleaner character
random funny events
3 themes
buttons for control
responsive design

what i learned from this project
how to link html, css and javascript together for one web app.
how to add advanced gradient colours and shadows in css.
how to add images using the <img> tag and use them again in js.
how to change the screen bg and make it look foggy and dirty.
how to switch themes cozy, cyberpunk, cartoon using css variables and js.
how to make buttions that reallyu do something when i ccik them clean, add, dirt, chnage theme.
how to use the canvas to drow fog, dust, drops and erase them with the mouse.
how to move a character on the screen and chnage its images for walking, cleaning, dancing, slipping, sleeping, drink coffe, angry mood etc.
how to show small speech bubbles with funny text using js.
how to make the layout work on desktop and mobile with responsive css./
